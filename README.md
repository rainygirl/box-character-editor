# box-character-editor

box-character-editor 는 [01410.coroke.net](https://01410.coroke.net) 서비스 구현 과정에서 마련된, KSC5601 선 특수문자`─│┌┐┘└├┬┤┴┼`로 상자를 그릴 수 있는 웹 에디터 코드입니다.

# 기술 사양

- HTML & CSS & JavaScript (Bundler 없음)

# 기동 방법

- editor.html 을 열면 됩니다.
